# Scientific Notation Scanner

Aplikasi web sederhana untuk memeriksa apakah sebuah angka ditulis dalam **Scientific Notation** (Notasi Ilmiah).

## Fitur

- Deteksi otomatis Scientific Notation (`e` atau `E`)
- Menu bar: **File**, **Edit**, **Run**, **About**
- Tombol **Run** di sebelah kanan input
- Hasil jelas dengan warna (Yes/No)
- Support tekan **Enter** untuk scan cepat
- Desain responsif dan modern

## Cara Menjalankan

1. Download atau buat 4 file berikut di satu folder:
   - `index.html`
   - `style.css`
   - `script.js`
   - `README.md`

2. Buka file `index.html` menggunakan browser (Chrome, Edge, Firefox, dll).

## Contoh Input Valid

- `6.022e23`
- `-1.23E-4`
- `9.81E+5`
- `1.0e0`
- `-4.56e-10`

## Teknologi

- HTML5
- CSS3
- Vanilla JavaScript
